# BookAPI

API to add books and authors to a mysql database. 

## Setup
To run this project:

```
1. Install Docker Compose
2. cd ../webshopAPI
3. touch .env 
4. copy content from .env.dist to .env
5. set environment variables in .env
6. docker-compose build
7. docker-compose up -d
```

## Technologies

* Flask 1.1.1
* Python 3.7
* Docker 
* MySQL
